#RBTreelib
**red-black binary search tree written in c**


##Usage

```
$cd $rbtreelib_dir/example;
chmod +x example_run.sh;
./example_run.sh
```

##Validate

```
$cd $rbtreelib_dir/test;
chmod +x test_run.sh;
./test_run.sh
```
